Programación

Me gusta mucho el entorno de la programación, los lenguajes que más suelo ocupar para realizar proyectos son Java en el IDE de Netbeans y C# en el IDE de Visual Studio.

Búsqueda de información

En cuanto a búsqueda de información, me gusta indagar en libros de gran calidad, sobre todo de bases de datos bibliográficas, puesto que allí existe información confiable que se puede utilizar para realizar trabajos de investigación.

Deportes

Entre mis deportes favoritos están el fútbol y el ajedrez, son deportes que me han encantado desde mi infancia.

Cultura

En este aspecto me llama mucho la atención las culturas precolombinas, su forma de organización y la tecnología que implementaban en aquella época. La música clásica es también de mi agrado.

Social

Aunque no suelo utilizar las redes sociales a menudo, me gusta tener cuentas para poder enterarme de los sucesos del mundo actual y como entretenimiento. Las redes que utilizo son Twitter y YouTube.

Juegos

Me gustan mucho los videojuegos relacionados a fútbol, carreras de autos y de habilidad mental. A futuro me gustaría poder crear un videojuego en 2D.

Algunos Lenguajes de programación más utilizados

Para empezar, HTML (Hypertext Markup Language) no es un lenguaje de programación, sino un lenguaje de marcado: su función es la de estructurar documentos digitales.

  • Java

    Java es un lenguaje de programación open source y multiplataforma que, gracias a su versatilidad, es adecuado para, prácticamente, cualquier proyecto.

  • JavaScript

    El lenguaje de scripts dinámico orientado a objetos no guarda relación con Java a pesar de su nombre, aunque ambos comparten el hecho de estar escritos en C.

  • PHP

    El preprocesador de hipertexto, más conocido por sus siglas PHP, es un lenguaje de scripting escrito en Perl y C. Se utiliza, principalmente, en la programación de páginas webyaplicaciones web dinámicas.

  • Python

    Python es un lenguaje de programación web de alto nivel basado en un código compacto, pero con una sintaxis fácil de entender. Python es también fácil de escribir porque, por ejemplo, los bloques no están separados con caracteres especiales, sino mediante sangrías.
